Concept

Arrays

An array is a data structure for storing more than one data item that has a similar data type. Array Representation:

array = [Element[0],Element[1],Element[2],Element[3],Element[5]]

Like lists, arrays are ordered, meaning each item has a numbered position.

Array name: necessary for easy reference to the collection of elements Data Type: necessary for type checking and data integrity Elements: these are the data values present in an array

Let's write array Syntax in python:

arrayName = array.array(typecode, [Initializers])

Here we show you how to use LISTS as ARRAYS.

Python has a separate module for handling arrays called array, which you need to import before you start working on them.
The array is versatile, and can serve as a useful tool in many different situations, but let’s just give one quick example.

Here’s an array of balance amount:

balance = array.array('i', [300,200,100])
